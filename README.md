## Описание проекта  

**Beauty Salon Bot** — это Telegram-бот, который:  
- показывает список услуг и мастеров,  
- ведёт базу клиентов и записей (PostgreSQL),  
- логирует все действия пользователей (MongoDB),  
- поддерживает импорт/экспорт CSV-файлов,  
- имеет CLI-режим для администратора,  
- в будущем будет интегрирован с **n8n** для автоматизации уведомлений и аналитики.

## Использование и установка зависимостей  
```bash
mvn clean install
Запуск бота
bash
mvn compile exec:java
Или собрать .jar и запустить:

bash
mvn package
java -jar target/bis-helper-bot.jar
Использование
Для клиентов (в Telegram):
Найдите бота по username (созданному у @BotFather)

Отправьте команду /start

Выберите "Услуги"

Выберите нужную услугу (например, Маникюр)

Выберите дату и время

Подтвердите запись

Пример взаимодействия:

makefile
Клиент: /start
Бот: 👋 Добро пожаловать!
     [Услуги]

Клиент: *нажимает "Услуги"*
Бот: Наши услуги:
     [Маникюр (60 мин)]
     [Педикюр (90 мин)]

*Дальнейший функционал в разработке*
Клиент: *выбирает "Маникюр"*
Бот: 📅 Выберите дату:
     [07.11] [08.11] [09.11]...

Клиент: *выбирает дату*
Бот: ⏰ Свободное время:
     [08:00] [09:30] [15:00]...

Клиент: *выбирает время*
Бот: ✅ Отлично! Вы записаны на 08.11.2025 в 15:00
🛠 Для администратора (CLI):
CLI позволяет экспортировать записи, импортировать услуги и просматривать данные в БД.

bash
mvn exec:java -Dexec.mainClass="ru.mybot.CLI"
Пример меню CLI:

markdown
=== Beauty Salon CLI ===
1. Экспорт записей
2. Импорт услуг
3. Показать услуги

Выбор: 2
Импортирована услуга: Маникюр (id=1)
Импортирована услуга: Педикюр (id=2)
Импорт завершён. Всего: 2
🧪 Тестирование

bash
use mybot_mongo

sql
